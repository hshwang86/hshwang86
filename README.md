## 🔭 현재 진행 중인 프로젝트

- 🚌 **MobilHi ERP** — 시내버스 운수업에 특화된 통합 ERP (배차, 급여, 인센티브 계약, 법규 준수 관리). 여러 운수 회사에 서비스 중. Django 5 · DRF · PostgreSQL · React.
- 📱 **현장 모바일 앱** — 운전자 음주측정 관리 등 현장 업무용 Flutter 앱. REST + JWT로 ERP 백엔드와 연동, Firebase 푸시 알림 지원.
- 🤖 **AI 자동화** — FastAPI + Claude/OpenAI 에이전트 기반 비즈니스 영상 제작 및 ERP 고객지원 자동화 파이프라인.
- 📈 **알고리즘 트레이딩** — Upbit/Binance 기반 암호화폐 자동매매 봇과 마켓 스크리너. ccxt, pandas-ta, 멀티 타임프레임 분석.
- 🎮 **게임 개발** — 서버 권위(server-authoritative) 실시간 온라인 게임, 웹 기반 경제 시뮬레이션 게임.
